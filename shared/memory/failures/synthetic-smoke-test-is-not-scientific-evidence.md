# Synthetic smoke-test gains are not scientific evidence

## Failure / caution
The current upgraded pipeline smoke test uses only 6 synthetic examples and includes preset mock gains in model state.

## Why it matters
Directionally correct rankings (raw < flat < hierarchical) validate pipeline wiring and reporting flow, but they do not validate the research claim.

## Durable lesson
Do not promote synthetic scaffold results as empirical support. The first real gate must be a small real-data pilot with matched summarizer, retriever, and answer-time token budget.
