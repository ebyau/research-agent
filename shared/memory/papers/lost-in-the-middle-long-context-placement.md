# Lost in the Middle (TACL 2024)

## Relevance
Useful as motivation that long-context models underuse relevant evidence when it is poorly placed or buried.

## Durable takeaway
Bigger context windows do not guarantee effective evidence use. This supports studying retrieved summaries as a better interface for small-model reasoning.

## Limitation for this project
It does not directly test retrieving generated intermediate summaries or compare flat vs hierarchical summary organization.
