# GraphRAG / hierarchical summary pipelines

## Relevance
Adjacent evidence that hierarchical aggregation can help answer global questions over large corpora.

## Durable takeaway
Hierarchy is plausible because organization of evidence matters, especially for compositional reasoning. This makes hierarchical summaries a credible intervention.

## Limitation for this project
Most evidence is system-level and confounded by graph indexing, planning, and larger pipelines. The clean missing ablation is flat vs minimal hierarchy under equal budgets for small LMs.
