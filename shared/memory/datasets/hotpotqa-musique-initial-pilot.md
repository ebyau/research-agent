# HotpotQA and MuSiQue for first pilot

## HotpotQA
- Strengths: standard multi-hop benchmark with supporting-facts annotations; good for EM/F1 and evidence-recall checks.
- Limitation: some questions are shortcut-prone and Wikipedia-only.

## MuSiQue
- Strengths: more composition-focused and designed to reduce shortcutting.
- Limitation: smaller ecosystem and may need more preprocessing.

## Durable use
Start with 300-500 dev examples per dataset for the first real pilot. Use both to separate easy shortcut wins from genuine compositional gains.
