# Top Lab Patterns Incorporated

This skill borrows the following patterns from strong research teams and modern agentic research systems:

## 1. Aggressive idea triage
- Generate multiple hypotheses
- Score them early
- Kill weak directions fast

## 2. Specialist roles instead of one omni-agent
- ideation
- literature
- gap mapping
- skepticism
- experiment design
- engineering
- analysis
- review
- memory capture

## 3. Human checkpoints at leverage points
- after critique
- after brief
- before expensive experiments
- before claiming publication-worthiness

## 4. Treat failures as assets
- write failed branches to memory
- store why they failed
- avoid repeating dead ends

## 5. Separate exploration from exploitation
- broad idea generation first
- deepen only top-ranked directions

## 6. Reviewer-aware thinking
- novelty is not enough
- baseline strength matters
- empirical rigor matters
- likely objections should be surfaced early
